Data Imputing Strategy

Missing values of 'steps' are being replaced by the mean for that 5-minute interval over all days.

for(i in 1:nrow(act)) {
  roww <- act[i,]
  if(is.na(roww$steps)) {
      roww$steps <- timeIntervalStepsMean[as.character(roww$interval)]  
      act[i,] <- roww
  }
}
